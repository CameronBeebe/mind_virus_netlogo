This model is a large modification of the built-in Netlogo model Virus on a Network from Wilensky.  It is adapted to model memes and mind viruses, which stems from ideas of biologist Richard Dawkins.

https://www.inf.fu-berlin.de/lehre/pmo/eng/Dawkins-MindViruses.pdf

Hosted here, but very slow compared to running locally.
https://cameronbeebe.github.io/Mind_Virus.html
